# 🏐 Weekly Volleyball Cup - 使用指南

这是一个和球友们分享比赛、记录精彩瞬间的平台。

## 快速开始

### 1 注册账号
- 访问网站，点击"注册"按钮
- 输入你的**邮箱地址**
~~~
（这是你的登录账号，我没有开启邮箱认证，所以按照邮箱格式填一个虚拟的也没关系）
~~~
- 设置密码（至少 6 位）
- 填写你的**用户名**（球友们会看到这个名字）
- 完成注册！

### 2️ 登录
- 使用注册时的**邮箱**和**密码**登录
- 登录后就能看到所有功能啦！

## 主要功能

### 查看赛事时间轴
- 点击"赛事时间轴"按钮
- 可以看到所有历史比赛和最近一周的比赛
- 使用筛选功能：
    - **全部赛事**：显示所有比赛
    - **最近一周**：只显示最近 7 天的比赛
    - **历史比赛**：查看已结束的比赛

### 查看比赛详情
- 在时间轴上点击任意比赛卡片
- 可以看到：
    - 本周主题（可以编辑哦）
    - 参赛队伍信息
    - 比赛规则
    - 结果公示
    - 照片和评论

### 管理参赛队伍
如果你是比赛组织者：
- 在比赛详情页可以添加参赛队伍
- 填写队伍名称、队员信息等
- 随时可以删除队伍

### 分享精彩瞬间
- 上传比赛照片
- 给照片添加文字说明
- 评论和互动

### 查看球友列表
- 在首页点击"查看用户列表"
- 看看都有哪些球友加入了我们
- 只有登录用户才能查看哦！

## 💡 小贴士

1. **关于编辑权限**
    - 所有登录用户都可以编辑比赛信息（协作编辑）
    - 但已结束的比赛不能修改
    - 请大家友好协作，不要随意删除别人的内容

2. **关于用户名**
    - 用户名注册后不能自己修改
    - 如果需要修改，请联系管理员

3. **关于照片上传**
    - 支持常见图片格式（JPG、PNG 等），可批量上传
    - 上传后可以添加文字说明
    - 每张图片设有评论区

4. **移动端使用**
    - 网站支持手机访问
    - 所有功能在手机上都能正常使用
